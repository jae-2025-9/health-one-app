# 건강 one app 제품 개요

## 한 줄 설명

건강 one app은 몸에 영향을 주는 생활 데이터를 기록하고 AI로 정리해, 사용자가 하루의 건강 상태와 주의할 점을 한 화면에서 볼 수 있게 하는 개인 건강 관리 앱입니다.

## 해결하려는 문제

건강 데이터는 걸음 수, 운동, 식사, 카페인, 수분, 수면, 약, 영양제, 화장품 성분처럼 여러 앱과 메모에 흩어져 있습니다. 건강 one app은 이 데이터를 하나의 시간축에 모아 사용자의 생활 패턴과 위험 신호를 보기 쉽게 만듭니다.

## v1 핵심 사용자 행동

1. 오늘의 활동량, 수면, 식사, 음용, 복용 기록을 확인합니다.
2. 식사 사진을 올려 칼로리와 영양 정보를 추정합니다.
3. 음료 라벨을 찍어 카페인, 당류, 열량을 기록합니다.
4. 약이나 영양제를 등록하고 복용 알림을 받습니다.
5. 약/영양제 조합의 주의 정보를 확인합니다.
6. 화장품 성분을 입력하거나 사진으로 올려 성분 분석 결과를 봅니다.
7. 일간/주간 리포트에서 수면, 활동, 섭취, 복용 패턴을 요약합니다.

## v1 기능 범위

| 영역 | 포함 기능 |
| --- | --- |
| 활동 | 걸음 수, 운동 시간, 활동 칼로리, 외부 Health API 업로드 |
| 수면 | 수면 시작/종료, 총 수면 시간, 수면 점수 또는 품질 메모 |
| 식사 | 사진 분석, 음식 항목, 칼로리, 탄수화물/단백질/지방 추정 |
| 음용 | 물, 커피, 캔음료, 카페인, 당류, 용량, 라벨 분석 |
| 약/영양제 | 품목 등록, 복용 로그, 복용 시간 알림, 상호작용 주의 정보 |
| 화장품 | 제품 등록, 성분 분석, 피부 타입별 주의 문구, 사용 로그 |
| AI 분석 | 사진/라벨/성분 분석, 일간/주간 요약, 신뢰도 저장 |
| 알림 | 복용 알림, 수분 섭취 알림, 사용자 설정 반복 규칙 |

## 안전 및 표현 원칙

- 앱은 의료 진단, 치료, 처방을 제공하지 않습니다.
- 약/영양제 상호작용 결과는 `low`, `medium`, `high`, `unknown` 위험도와 근거 요약으로 제공합니다.
- 위험도가 높거나 근거가 불확실한 경우 전문가 상담 권고 문구를 반드시 포함합니다.
- Apple HealthKit과 Samsung Health 데이터는 사용자 동의 후 모바일 앱에서 읽어 서버에 동기화합니다.

## 팀 세션에서 말할 핵심

건강 one app은 "내 몸에 들어오고 닿고 영향을 주는 모든 생활 인자"를 하나의 건강 이벤트 타임라인으로 관리하는 앱입니다. ERD는 공통 `health_events` 테이블을 중심으로 각 도메인 세부 테이블을 연결하고, API는 테이블 기준이 아니라 사용자의 행동 기준으로 나눕니다.
